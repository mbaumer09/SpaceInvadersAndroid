# SpaceInvadersAndroid

A classic Space Invaders arcade game reimplemented for Android.

## Features

- Classic Space Invaders gameplay
- Touch controls - tap and drag to move and shoot
- Progressive difficulty with multiple levels
- Score tracking and lives system
- Fullscreen portrait mode

## How to Play

- Touch and drag left/right to move your ship
- Your ship automatically fires when you touch the screen
- Destroy all enemies to advance to the next level
- Avoid enemy fire and prevent enemies from reaching the bottom
- You start with 3 lives

## Game Elements

- **Player Ship**: Your green spaceship at the bottom
- **Enemies**: Green invaders that move side-to-side and down
- **Cyan Bullets**: Your projectiles
- **Red Bullets**: Enemy projectiles

## Building

This is a standard Android project built with Gradle. Open in Android Studio and run, or build from command line:

```bash
./gradlew assembleDebug
```

## Requirements

- Android SDK 21 (Lollipop) or higher
- Kotlin support
